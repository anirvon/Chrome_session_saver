# Privacy Policy Draft for Session TXT/JSON Saver

Last updated: [DATE]

Session TXT/JSON Saver is a Chrome extension that lets users export their open Chrome windows, tabs, and tab groups to a local session file, and import that file later.

## Data the extension handles

The extension may read the following information from currently open Chrome windows and tabs when the user clicks the export button:

- Tab URLs.
- Tab titles.
- Tab order.
- Window metadata such as size and position.
- Tab group metadata such as title, color, and collapsed state.
- Tab state such as pinned, active, and muted.

This information is written into a local file chosen/downloaded by the user, such as `sessions.json` or `sessions.txt`.

## Data the extension does not collect

The extension does not collect, read, or store:

- Page contents.
- Cookies.
- Passwords.
- Payment information.
- Form entries.
- Full browsing history.
- Analytics or telemetry.

## Data storage

The extension does not store session data on a server. Exported session files are saved locally by the user's browser. Users are responsible for storing, moving, deleting, or sharing those files.

## Data sharing

The extension does not sell, transmit, or share user data with third parties. The extension does not use analytics, tracking scripts, advertising SDKs, or remote servers.

## Network access

The extension does not make network requests for its own operation. It only asks Chrome to open URLs during session import, based on the contents of the local session file selected by the user.

## Limited use statement

The use of information received from Chrome extension APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. Data is used only to provide the user-facing session export and import functionality.

## User control

Users control when a session is exported and which file is imported. Users can delete exported files at any time using their operating system's file manager.
